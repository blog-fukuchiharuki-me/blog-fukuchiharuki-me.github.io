# ブログに書くこと

レビューやレポートなど、調べたり考察したりしたことを書く。記事の内容は客観的であるように心がける。より主観を伴うことは [note](https://note.com/fukuchiharuki) に書く。ただし、経験を振り返るようなことはこちらに書く。
